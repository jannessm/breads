# Bread Banking Planner

## Overview
A simple web application to help bakers plan and track bread recipe stages with precise timing.

## Features
- Add recipe stages with custom names and durations
- Calculate consecutive start times
- Save and load recipes using local storage
- Delete existing recipes

## How to Use
1. Enter a recipe name
2. Add stages with their names and durations
3. Set a start time to calculate stage timings
4. Save the recipe for future reference

## Technologies
- HTML5
- JavaScript
- Local Storage API

## Deployment
This app is deployed on GitHub Pages and can be accessed directly in the browser.
